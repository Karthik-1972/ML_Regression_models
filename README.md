# Machine Learning Regression Models

This repository contains various implementations of regression algorithms used in machine learning. The primary goal is to provide a comprehensive understanding of different regression techniques and their performance on standard datasets.

## Algorithms Included
- Linear Regression
- Ridge Regression
- Lasso Regression
- Polynomial Regression
- Support Vector Regression
- Decision Tree Regression
- Random Forest Regression

## Requirements
- Python 3.x
- Libraries: NumPy, Pandas, Scikit-learn, Matplotlib

## Getting Started
To get started with the models:
1. Clone the repository:
   ```bash
   git clone https://github.com/Karthik-1972/ML_Regression_models.git
   ```
2. Navigate to the directory:
   ```bash
   cd ML_Regression_models
   ```
3. Install the required libraries:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the desired model script.
